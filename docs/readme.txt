Graphing Application 1.0

Reads point data from a text(txt) file and displays that data on a graph.

Use the UI (mouse) or keyboard commands to configure the graph:
- Text file should be in executable directory and named "point_data.txt"
- Point data format int or float and is space and line delimited:
    <x> <y>
    <x> <y>

Keyboard commands:
 + Increase Scale
 - Decrease Scale
 Num (8) Incrase Size
 Num (2) Decrease Size
 Num (0) Refresh Point Data (Can change txt file while running)
 Num (7) Toggle Show Coordinates
